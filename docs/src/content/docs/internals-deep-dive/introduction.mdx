This knowledge base is dedicated to everything [Knockout City](https://www.knockoutcity.com/)
internal related. It disects the different formats the game uses as well as
providing a helping hand to developers for mod APIs.

Currently just a few bits and pieces have been reverse-engineered which will
hopefully be expanded opon in the near future.

Make sure to visit the <Link href='internals-deep-dive/getting-started'>Getting Started</Link> page to dive
into the world of [Knockout City](https://www.knockoutcity.com/) internals.

:::tip[Symbols explained]
Through out the knowledge base you might encounter the following symbols:

- `???`: Purpose / usage yet unknown.
