The Text Component is used for display text on screen.

## Structure

```typescript
type TextComponent = {
  /**
   * The VTable Reference
   */
  vtable: GUID;
  /**
   * The unique GUID for the component
   */
  $guid: GUID;
  /**
   * The name of the component
   */
  component_name?: string;
  /**
   * If all letters should be forced to be uppercase
   */
  force_uppercase: boolean;
  /**
   * The font size
   */
  initial_font_size: number;
```
