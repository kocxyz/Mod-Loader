## Details

**Table**: `commerce_accessories`  
**Description**: This table contains information about the accessory items in the game.

## Fields

- `item` - The item GUID
- `is_crew` - If the item is a crew item
- `is_consumable` - If the item is a consumable
- `ui_name` - The display name of the item
- `rarity` - The rarity of the accessory
- `accessory_type` - The type of the accessory
- `platform_restriction` - The platform the item is restricted to
